# Hexlet DevOps educational project.
A project to learn how docker and docker-compose work.

## Required
- docker
- docker-compose

## For develop
Use: `make compose`
Don't forget to create `.env` file with necessary variables. Look at `.env.example`

## For testing production image
Use: `make test-image`
